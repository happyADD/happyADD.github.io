# 三自由度稳定云台

> 基于 STM32 的高精度自定义控制器云台系统，实现电机闭环控制

## 项目概述

本项目实现了基于 STM32F103C8T6 的三自由度云台控制系统，通过自定义手柄控制器实现对云台的精准控制。系统采用 **PID 双闭环控制算法** 保证角度控制精度，并集成 MPU6050 陀螺仪实现姿态反馈补偿。

## 核心技术栈

### 🔧 硬件平台
| 模块   | 型号/规格                     | 作用           |
| ---- | ------------------------- | ------------ |
| 主控芯片 | STM32F103C8T6 (Cortex-M3) | 系统核心控制器      |
| 驱动电机 | MG310 编码器直流减速电机 ×2,SG20舵机 | 云台三自由度执行机构   |
| 传感器  | MPU6050 六轴陀螺仪             | 姿态反馈与角度补偿    |
| 通信模块 | HC-05 蓝牙模块                | 实现控制器与云台无线通信 |
| 辅助显示 | 0.96寸 OLED 屏              | 实时显示双轴角度数据   |

### 💻 软件与工具链

- 开发环境：Keil MDK5 + STM32CubeMX（HAL库自动生成）
- 控制算法：
  • 外环：位置环 PID（角度精准控制）
  • 内环：速度环 PID（抑制超调与振荡）
- 通信协议：自定义串口协议（有线/蓝牙双模）
- 版本管理：Git + Gitee（完整提交记录，20+ commits）
- 调试工具：串口助手（实时角度监控）、逻辑分析仪（PWM波形分析）


## 功能实现亮点

 **无线自定义控制器**  
采用双 STM32 架构：
- 主控端：接收手柄输入 → 蓝牙发送控制指令
- 云台端：接收指令 → 双电机协同运动 → OLED 实时反馈角度

**多任务实时调度**  
基于定时器构建三任务架构：
```c
Task 1: IMU   // 执行IMU 位姿解算和 PID 运算
Task 2: SendFloats    // 事件驱动，解析控制指令
Task 3: Oled    // 刷新角度显示
```

 **工程化开发流程**  
- 使用 STM32CubeMX 配置时钟树、GPIO、TIM、USART 外设
- 通过 Git 进行版本迭代管理
- 自主设计接线方案与机械固定结构，实现稳定可靠运行
